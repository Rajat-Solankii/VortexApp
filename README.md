# Vortex 🌌

Vortex is a cutting-edge Android media streaming application built with **Jetpack Compose**. It combines a stunning **Glassmorphic UI** with powerful features like automated update tracking and a high-performance custom player to deliver a premium viewing experience.

---

## ✨ Key Features

### 🎨 Modern Glassmorphic UI
*   **Aesthetic Design**: A beautiful, translucent interface that feels modern and fluid.
*   **Dynamic Theming**: Smooth transitions and blur effects throughout the app.
*   **Responsive Layouts**: Optimized for various screen sizes with Jetpack Compose.

### 🎬 Advanced Streaming & Playback
*   **Custom Player**: High-quality video playback with intuitive controls.
*   **Picture-in-Picture (PiP)**: Keep watching your favorite content while multitasking.
*   **Smart Resuming**: Automatically track and resume your watch history across sessions.
*   **Auto-Play**: Seamlessly transition to the next episode in a series.

### 🔍 Intelligent Discovery
*   **Smart Search**: Quickly find Movies, TV Shows, and Anime with real-time results.
*   **Onboarding**: Tailored content recommendations based on your initial genre preferences.
*   **Personalized Library**: Easily manage your watchlist and recently watched content.

### 🔄 Smart Updates & Monitoring
*   **Automated Updates**: Stay up-to-date with automatic version checking via GitHub.
*   **Telegram Integration**: Real-time device reporting and feedback system for administrative monitoring.

---

## 📦 Setup & Updates

Vortex is designed to be self-maintaining. The app periodically checks `update.json` in this repository to ensure you are running the latest version with the newest features and security patches.
